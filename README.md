# Define Clinic Website v2

A redesigned, Ant-free Next.js app-router website for Define Skin, Hair & Laser Clinic.

## Stack
- Next.js (modern app router)
- React
- TypeScript
- CSS Modules

## Run
```bash
nvm use
npm install
npm run dev
```

Open `http://localhost:3000`.

## Node Version
- Required: Node.js `>=20.9.0`
- Project default via `.nvmrc`: `20.11.1`

## Pages
- `/`
- `/home`
- `/services`
- `/about`
- `/book-appointment`
- `/contact-us`

## Content source
All clinic business content is centralized in `content/clinic.ts`.
